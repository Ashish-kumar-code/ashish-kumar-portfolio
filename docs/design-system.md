# Design System

Version: 1.0

---

# Design Philosophy

The portfolio should feel like a modern SaaS product rather than a traditional personal website.

The visual language emphasizes simplicity, professionalism, clarity, and engineering excellence.

Every visual element should support the content and improve usability.

The design should communicate confidence without unnecessary complexity.

---

## Core Principles

### Clarity First

Information should be immediately understandable.

Avoid decorative elements that compete with content.

---

### Professional Aesthetic

Inspired by:

- Linear
- Vercel
- Stripe
- GitHub
- Raycast

---

### Consistency

Spacing, typography, colors, animations, and components should follow one unified system.

---

### Performance

Every visual effect should justify its existence.

Avoid excessive animations.

---

### Accessibility

Accessibility is a design requirement, not an afterthought.

Keyboard navigation

Proper contrast

Semantic structure

Visible focus states
